API 接口列表
==================================================

GET    /                                        Root Path
POST   /api/annotations/validate                验证标注数据
POST   /api/annotations/validate-partial        验证部分标注数据
GET    /api/annotations/{task_id}/documents     获取任务文档列表
GET    /api/annotations/{task_id}/documents/{document_id}/annotation 获取标注数据
POST   /api/annotations/{task_id}/documents/{document_id}/annotation 保存标注数据
GET    /api/annotations/{task_id}/documents/{document_id}/content 获取文档内容
GET    /api/annotations/{task_id}/documents/{document_id}/form-config 获取标注表单配置
GET    /api/annotations/{task_id}/documents/{document_id}/review 获取复审数据
POST   /api/annotations/{task_id}/documents/{document_id}/review 提交复审
POST   /api/annotations/{task_id}/documents/{document_id}/submit 提交文档标注
GET    /api/annotations/{task_id}/progress      获取任务进度统计
DELETE /api/annotations/{task_id}/{document_id} 删除标注数据
GET    /api/annotations/{task_id}/{document_id} 获取标注数据
POST   /api/annotations/{task_id}/{document_id} 保存标注数据
PUT    /api/annotations/{task_id}/{document_id} 更新标注数据
POST   /api/auth/change-password                修改密码
POST   /api/auth/login                          用户登录
GET    /api/auth/me                             获取当前用户信息
POST   /api/auth/refresh                        刷新访问令牌
POST   /api/auth/register                       用户注册
GET    /api/files/                              获取文件列表
GET    /api/files/download/batch                批量下载文件
GET    /api/files/my-files                      获取我的文件
POST   /api/files/upload                        上传文件
POST   /api/files/upload/batch                  批量上传文件
DELETE /api/files/{file_id}                     删除文件
GET    /api/files/{file_id}/download            下载文件
GET    /api/files/{file_id}/preview             预览文件
GET    /api/files/{file_id}/validate            验证模板文件
GET    /api/tasks/                              获取任务列表
POST   /api/tasks/                              创建任务
GET    /api/tasks/statistics                    获取任务统计
DELETE /api/tasks/{task_id}                     删除任务
GET    /api/tasks/{task_id}                     获取任务详情
PUT    /api/tasks/{task_id}                     更新任务
PUT    /api/tasks/{task_id}/documents/{document_id}/status 更新文档状态
POST   /api/tasks/{task_id}/export              导出任务数据
GET    /api/tasks/{task_id}/progress            获取任务进度详情
GET    /api/tasks/{task_id}/template/fields     获取任务模板字段
GET    /api/users/                              获取用户列表
DELETE /api/users/{user_id}                     删除用户
GET    /api/users/{user_id}                     获取用户详情
PUT    /api/users/{user_id}                     更新用户信息
GET    /health                                  Health Check
